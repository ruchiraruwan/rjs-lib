RJS Library

This is a lightweight JavaScript library which supports multiple features.

  • Highlight the search text of the current page
  • Password strength indicator
  • Display the meaning of the selected word

Feature 1: Highlight the Search Text of the Current Page

Live Demo, you can type whatever text in the search box and it will highlight the text if it exist in the current page.

You can easily add this feature with the following code lines into HTML page and include rjs-lib.js
<input type="text" id="rjs-search-text" placeholder="Search on this page...">
<button id="rjs-search-btn">Search</button>
<span id="rjs-search-result"></span>
